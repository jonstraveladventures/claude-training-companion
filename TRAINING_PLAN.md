# Training plan (EXAMPLE — replace with your own)

> This is a sample plan to show the format. Claude reads it to answer "what's
> today's session?" and to cross-check what you log. Edit freely.

## Goals

1. **Strength:** progress a main lift over a multi-week block (e.g. squat 5×5).
2. **Endurance:** build an aerobic base on a polarised model (mostly easy, a little hard).
3. **Consistency:** log every session so progress is visible over time.

## Weekly structure (example)

| Day | Focus | Time |
|---|---|---|
| Mon | Upper strength | 60 min |
| Tue | Easy run (HR-capped) | 45 min |
| Wed | Lower strength (main-lift focus) | 70 min |
| Thu | Optional accessory / mobility | 30 min |
| Fri | Rest | — |
| Sat | Quality run (intervals / threshold) | 45 min |
| Sun | Long easy run | 60–90 min |

## Main-lift progression (example)

Add a small increment each block; consolidate before advancing; deload periodically.

| Week | Top set | Notes |
|---|---|---|
| 1 | 95 × 5×5 | |
| 2 | 100 × 5×5 | first jump |
| 3 | 100 × 5×5 | consolidate |
| 4 | 80 × 3×5 | deload |
| 5 | 105 × 5×5 | |

**Stall rule:** if you miss a top set, repeat that weight before advancing.

## Running zones (example — set yours)

| Zone | HR | Use |
|---|---|---|
| Z1 | ≤ 134 | recovery |
| Z2 | 134–148 | endurance (most volume here) |
| Z3 | 148–163 | grey zone (limit) |
| Z4 | 163–178 | threshold |
| Z5 | > 178 | VO2max |

**Polarisation target:** ≥ 75% of running time in Z1+Z2, ≤ 20% in Z4+Z5.
